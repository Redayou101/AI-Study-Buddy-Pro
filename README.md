# 🎓 AI Study Buddy Global Pro v2.0

An advanced AI-powered educational tool designed to help students analyze, summarize, and extract key insights from their learning materials efficiently. This project aims to revolutionize how students interact with their study content through automation and intelligence.

--

## 🚀 Key Features

* **✨ Intelligent Summarization:** Automatically breaks down long articles into 3 core executive points to save time.
* **🔑 Keyword Extraction:** Identifies the most important concepts and terminology using custom string analysis logic.
* **📊 Smart Reporting:** Generates a professional "Study Report" including timestamps, word counts, and organized insights.
* **🛡️ Input Validation:** Built-in safeguards to ensure high-quality analysis by validating text length and format.

---

## 🛠️ Technology Stack

* **Language:** Python 3.x
* **Concepts:** * String Manipulation & Cleaning
    * List Comprehension (Optimized processing)
    * Functional Programming (Clean modular code)
    * Data Dictionaries (Structured output)

## 💻 How to Run

1. Clone the repository.
2. Run `main.py` using a Python 3 interpreter.
3. Paste your text and get your instant AI report!
